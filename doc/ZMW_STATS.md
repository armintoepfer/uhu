<h1 align="center">
    zmw_stats
</h1>

Extracts the following metrics for each ZMW to standard out:
 - ZMW ID
 - Number of counts for each CX tag between 0 - 15
 - Forward barcode
 - Barcode Score
 - Number of Subreads
 - Subread Mean Length
 - Subread Median Length
 - Subread Length Standard Deviation
